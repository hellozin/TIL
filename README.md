## 정책

## 고민

- 앞으로 뭘 공부할까, 어떻게 공부할까도 계속 고민해야 하지만 지금까지 배우고 사용하는 것들을 더 확실하게 정리해보자.
- 주변에서 컨퍼런스 발표를 하고 배울게 많은 글을 쓰는 사람들은 어떤 내용을 공부하고 어떤 과정을 거쳐 그런 내용을 정리하게 되었을지 생각해보자.

- Java language
  - Best practice
  - JVM
  - GC

  - Spring framework
  - Cache
  - DB
    - DB의 종류와 장,단점

  - 성능 측정 및 최적화

    - 트래픽 분산
  - 고가용성
